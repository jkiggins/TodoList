To deploy the server first the user must

	1) download the djangoTutorial.zip file. 
	2) unzip the project into a directory of your choosing
	3) open a cmd window in the current directory	
	4) Type: python manage.py runserver

This assumes you have the correct python and django versions outlined in the requirements section installed and configured correctly.

To view the todo list application navigate to http://127.0.0.1:8000/todo/ after the above steps are followed.
